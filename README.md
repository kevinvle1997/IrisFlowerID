# IrisFlowerID


![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)

IrisFlowerID is a program written in Python and utilizes several libraries to build the model. It takes in several Iris structures and attempts to recognize and distinguish the structures from eachother using several different generated models


### Installation

Install the requirments.txt and run main.py 


And of course IrisFlowerID itself is open source with a public repository
 on GitHub.
